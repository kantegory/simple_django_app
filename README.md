# Simple Django App

1. Make git clone and go to project dir

```bash
git clone https://github.com/kantegory/web.git
mv web/lab01 ../ && cd lab01/django_project_dobryakov
```

2. Install all requirments
```bash
make init
```

3. Make migrations
```bash
make migrate
```

4. Run dev server
```bash
make run
```

*Notice:* if you need update requirments:
```bash
make requirments
```
